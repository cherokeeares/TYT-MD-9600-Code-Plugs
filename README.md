# TYT-MD-9600-Code-Plugs
Cherokee County ARES TYT MD-9600 Code Plugs

Before you load the code plug to your radio, you need to input your call sign and DMR ID.


MD-9600 Config File Changes that you MUST make in the software:

General Settings -> Intro Screen Line 1 -> Enter Callsign

General Settings -> Intro Screen Line 2 -> Enter DMR ID

General Settings -> Radio Name -> Enter Callsign

General Settings -> Radio ID -> Enter DMR ID
